criação de index
db.students.createIndex({student_id:1});

criação de idenx composto
db.students.createIndex({class:1, student_name:1});

para consultar os indexs de uma tabela
db.students.getIndexes();

Excluir index
o index _id nao pode ser exclui porque é default.
para excluir um index qualquer essa é sintaxe:
db.students.dropIndex({student_id:1});
